# Customer Behavior Analysis

## Project Overview

This project focuses on analyzing customer transactions and behavior to identify customer segments, purchasing trends and churn risks.

The analysis was performed using Python in Jupyter Notebook and includes data cleaning, feature engineering, customer segmentation and visualizations.

---

## Objective

The objective of this project is to:

* Clean and prepare customer data
* Analyze customer purchase behavior
* Segment customers into groups
* Visualize trends and retention
* Generate actionable business recommendations

---

## Dataset

Dataset Name:

Customer Behavior Analysis Dataset

Source:

Kaggle

Dataset includes:

* Customer details
* Product information
* Purchase history
* Customer churn information

## Project Workflow

### Step 1 — Data Loading

Load dataset into Jupyter Notebook.

### Step 2 — Data Cleaning

Remove missing values and duplicates.

### Step 3 — Feature Engineering

Create customer spending categories.

### Step 4 — Customer Segmentation

Group customers into:

* High Value
* Medium Value
* Low Value

### Step 5 — Visualization

Generate charts for:

* Customer Segments
* Purchase Patterns
* Retention Trends

### Step 6 — Business Insights

Provide recommendations for engagement improvement.
