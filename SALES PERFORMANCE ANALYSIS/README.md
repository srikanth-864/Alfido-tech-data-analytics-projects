# Sales Performance Analysis

## Project Overview

This project focuses on analyzing sales performance across regions, categories and time to identify business trends and opportunities.

The analysis was performed using Python and Jupyter Notebook and includes data cleaning, KPI calculation, visualization and business recommendations.

The project helps understand product performance, seasonality and regional sales contribution.

---

## Objective

The objective of this project is to:

* Analyze sales performance across regions
* Evaluate category-wise sales contribution
* Identify best and worst performing products
* Understand sales seasonality
* Generate business recommendations

---

## Project Workflow

### Step 1 — Data Loading

Load dataset into Jupyter Notebook.

### Step 2 — Data Cleaning

Remove missing values and duplicates.

### Step 3 — KPI Creation

Calculate:

* Revenue
* Profit Margin
* Average Order Value
* Total Orders

### Step 4 — Feature Engineering

Create Month and Year features from order date.

### Step 5 — Visualization

Generate charts for:

* Monthly Sales Trend
* Category Performance
* Regional Performance
* Product Analysis

### Step 6 — Business Insights

Identify patterns and provide recommendations.

## Key Findings

* Sales vary across regions.
* Categories contribute differently.
* Monthly seasonality affects performance.
* Top products drive major revenue.
* Low-performing products require improvement.

## Recommendations
